Angular がテンプレート・コントローラ・モデルの情報をもとに変換し、 ブラウザに読み込まれ描画される DOM のことを Angular ではビューといいます。

Angular の MVC 実装では、ビューはモデルとコントローラについての情報を持っています。 ビューには双方向データバインディングが行われているモデルが分かっています。 また、 ngController や ngView ディレクティブ、 {{someControllerFunction()}} の形式のバインディングを通して、 ビューはコントローラの情報を得ています。 このような実装なので、 ビューは関連付けられたコントローラ関数に含まれる関数を呼び出すことができるのです。
