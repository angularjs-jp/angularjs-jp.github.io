これまで長い間、サービスは一般的にサーバサイドで使用されていましたが、 Angular のサービスとは、 Angular によって、 サーバサイドからクライアントサイドの Web アプリケーションへと持ち込まれた機能のことを指します。 Angular アプリケーションにおけるサービスは、 依存性注入 (DI) によってつなぎ合わされる、代用可能なオブジェクトです。
